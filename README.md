## Development Conveniences

This project stays intentionally barebone, but includes a formatter setup for consistency.

### Formatter

- Tool: Prettier
- Config: `.prettierrc`
- Ignore list: `.prettierignore`
- Editor consistency: `.editorconfig`

### Install Dependencies

```bash
npm install
```

### Format Everything

```bash
npm run format
```

### Check Formatting (No Changes)

```bash
npm run format:check
```
